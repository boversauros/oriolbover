# My Personal Site 

This is my personal site, built using Next.js and powered by the GitHub API.

## Features

- Project list pulled from GitHub API
- Clean, minimalistic design

## Installation

To run this site locally, you'll need to have Node.js and npm installed. Then, clone this repository and run the following commands in your terminal:

```
# Install dependencies

npm install

# Start development server

npm run dev
```

## Configuration

To fetch your GitHub projects, you'll need to set up a personal access token with the repo scope in your GitHub account. Then, create a .env.local file in the root directory of this project and add your token like this:

```
GITHUB_TOKEN=your_token_here
```
